# Dotfiles

A set of configuration files for MacOS to set up a development environment

## Usage

```sh
./install.sh
```

## Development dependencies

* Homebrew
* Git
* Vim
* zsh
* tmux

## Configuration files (dotfiles)

* .zshrc
* .vimrc
* .tmux.conf

## Other misc dependencies

* Powerline fonts for oh-my-zsh
